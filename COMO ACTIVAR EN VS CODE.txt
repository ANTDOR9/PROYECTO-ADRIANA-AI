Opción 1 — Doble clic (más fácil)

Abre el explorador de Windows
Ve a D:\PROYECTO-ADRIANA-AI-main\
Doble clic en iniciar.bat


Opción 2 — Desde Git Bash
bashcd /d/PROYECTO-ADRIANA-AI-main
source venv/Scripts/activate
python adriana_ui.py

Opción 3 — Desde VS Code

Abre VS Code
Presiona Ctrl+` para abrir la terminal
Ejecuta:

bashsource venv/Scripts/activate
python adriana_ui.py
